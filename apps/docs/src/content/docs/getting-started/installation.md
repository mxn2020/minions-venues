---
title: Installation
description: Install Minions Venues packages
---

## TypeScript / Node.js

```bash
npm install @minions-venues/sdk minions-sdk
```

## Python

```bash
pip install minions-venues
```

## CLI

```bash
npm install -g @minions-venues/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
