# Minions Venues

Tent definitions, capacities, reservation rules, and opening schedules for Oktoberfest

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-venues/sdk`) — Core library for Node.js
- **Python SDK** (`minions-venues`) — Python bindings
- **CLI Tool** (`@minions-venues/cli`) — Command-line interface
- **Web App** — Interactive playground at [venues.minions.wtf](https://venues.minions.wtf)
